# music

## 项目使用管理工具为yarn
```
yarn
```

## yarn的常用命令

### 依赖下载命令
```
yarn install
```

### 编译和热更新（用于开发环境）
```
yarn serve
```

### 编译并压缩（用于生产环境）
```
yarn build
```

### 检查并修复源文件
```
yarn lint
```

### 检查 webpack 配置
```
yarn inspect
```
